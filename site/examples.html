Examples

Command Examples

Practical examples for all current commands. All examples assume jlcpcb is installed and available on your PATH.

Basic search

jlcpcb search "10k 0402 resistor"

jlcpcb search "100nF capacitor" --basic-only

jlcpcb search "LED" --preferred-only --min-stock 1000

jlcpcb search "STM32" --pages 3 --limit 50

search queries the live JLCPCB catalog and caches results locally. Use --basic-only to restrict to JLCPCB basic parts (lower assembly cost). Use --preferred-only for JLCPCB preferred parts.

Attribute filtering

jlcpcb search "0402 resistor" --attr "Resistance >= 10k"

jlcpcb search "0402 capacitor" --attr "Voltage Rated >= 25"

jlcpcb search "MOSFET SOT-23" \
  --attr "Drain Source Voltage (Vdss) >= 30" \
  --attr "Continuous Drain Current (Id) >= 5"

jlcpcb search "LDO 3.3V" --max-price 0.50 --basic-only

Operators: >= <= > < = !=. Multiple --attr flags are ANDed. SI prefixes are supported: 10k, 100n, 4.7u, 1M.

Output formats

Most commands accept --format table|json|csv|markdown.

Search as JSON

jlcpcb search "10k resistor" --format json
{
  "status": "ok",
  "command": "search",
  "count": 2,
  "results": [
    {
      "lcsc_code": "C8287",
      "mfr_part": "RC0402FR-0710KL",
      "manufacturer": "YAGEO",
      "package": "0402",
      "stock": 500000,
      "price_qty1": 0.0037
    }
  ]
}

BOM as JSON

jlcpcb bom --format json
{
  "status": "ok",
  "command": "bom",
  "data": [
    {
      "ref": "R1",
      "lcsc": "C8287",
      "quantity": 2,
      "warnings": []
    }
  ]
}

Note: bom uses "data" instead of "results".

Local cache queries

jlcpcb query

jlcpcb query "resistor" --package 0402

jlcpcb query --attr "Resistance >= 1k" --attr "Resistance <= 100k" --basic-only

query searches only the local cache — no network required. Useful for scripting and offline filtering.

Inspect and compare

jlcpcb fetch C8287
jlcpcb info C8287

jlcpcb fetch C8287 C25900
jlcpcb compare C8287 C25900

jlcpcb fetch C8287 --force

fetch caches one or more parts. --force refreshes even if the part was recently cached. info shows full specs, attributes, and links. compare shows a side-by-side table.

Datasheets

jlcpcb fetch C347356
jlcpcb analyze C347356 \
  --prompt "Summarize power rating and operating temperature range"

jlcpcb analyze C347356 --model "anthropic/claude-sonnet-4.6"

jlcpcb datasheet C347356 --pdf -o docs/datasheets/

jlcpcb datasheet C347356 --pdf --summary -o docs/datasheets/

analyze sends the datasheet PDF to OpenRouter and returns markdown. datasheet --summary writes a .md file alongside the PDF. Both require an OpenRouter API key.

Project workflow

Start a new project

mkdir my-board && cd my-board
git init
jlcpcb init --name "my-board" --description "Motor driver board"

Research and select components

jlcpcb search "100nF 0402 X7R"
jlcpcb fetch C1525
jlcpcb info C1525

jlcpcb select C1525 --ref C1 --qty 1 --notes "Bypass cap"
jlcpcb select C1525 --ref C2 --qty 1 --notes "Bypass cap"
jlcpcb select C347356 --ref U2 --qty 1 --notes "LED driver"

BOM review

jlcpcb bom
jlcpcb bom --check
jlcpcb bom --format csv
jlcpcb bom --format markdown
jlcpcb status

Edit references

jlcpcb relabel R1 R3
jlcpcb deselect C2

Work from outside the project directory

jlcpcb --project ~/Projects/my-board bom

export JLCPCB_PROJECT=~/Projects/my-board
jlcpcb status

Commit the result

git add .jlcpcb/project.yaml
git commit -m "Select initial BOM components"

Database commands

jlcpcb db stats

jlcpcb db clear

The shared cache lives at ~/Library/Application Support/jlcpcb/parts.db on macOS or ~/.local/share/jlcpcb/parts.db on Linux. db clear wipes all cached parts — they can be re-fetched at any time.
