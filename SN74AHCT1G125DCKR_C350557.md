# SN74AHCT1G125DCKR Technical Summary

## Overview
Single-channel, non-inverting, 3-state bus buffer/line driver in SC-70-5 (DCK) package. Designed for TTL-to-5V level translation and bus interface applications.

## Typical Application Circuit

**3.3 V → 5 V Level Translation:**

```
3.3V Logic ──► A (Pin 2) ──► [SN74AHCT1G125] ──► Y (Pin 4) ──► 5V Logic/Load

                              V_CC (Pin 5) ──┬── Regulated 5V
                                            C1 (0.1µF to GND)

                              GND (Pin 3) ──── GND

                              OE (Pin 1) ──── Control Signal or GND (for always-enabled)
                                         (Pull to V_CC via resistor for Hi-Z at power-up)
```

### Recommended Component Values
| Component | Value | Purpose |
|-----------|-------|---------|
| C1 (bypass cap) | **0.1 µF** ceramic | V_CC decoupling; place as close to Pin 5 as possible |
| C2 (optional parallel) | **1 µF** ceramic | Broadband noise rejection |
| R_PU (OE pull-up) | Value per sink current of driver | Ensures Hi-Z during power-up/power-down |

## Important Design Notes

- **Power-Up Safety:** Connect OE (Pin 1) to V_CC via pull-up resistor during power-up/power-down to maintain output high-impedance and prevent bus contention.
- **3.3V→5V Translation:** TTL-compatible input thresholds (V_IH = 2.0 V min) allow direct interfacing with 3.3 V CMOS/TTL logic while outputting 5 V levels.
- **No Floating Inputs:** All unused inputs must be tied to V_CC or GND. Floating inputs cause undefined states and excess I_CC.
- **Bus Contention:** Avoid driving multiple enabled outputs onto a common bus simultaneously; continuous output current limit is ±25 mA per output, ±50 mA total per device.
- **Output Loading:** Do not pull outputs above V_CC. Observe ΔI/ΔV input transition rate ≤ 20 ns/V.
- **I_CC Penalty:** Each input held at a TTL intermediate voltage (e.g., 3.4 V) adds ~1.5 mA to I_CC (ΔI_CC specification).
- **Package:** SC-70-5 body is 2.00 × 1.30 mm; θ_JA = 287.6°C/W — thermal management required at maximum output current.
- **MSL Rating:** Level-1-260°C-UNLIM (no moisture precautions required for soldering).
